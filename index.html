📖 Introducción

El Sistema de Gestión de Inventarios es una aplicación web desarrollada para la administración integral de productos de seguridad electrónica. Este manual proporciona una guía completa para utilizar todas las funcionalidades del sistema de manera eficiente.

Versión: 1.0.0
Fecha: Octubre 2025
Tecnología: AdminLTE Framework

✨ Características Principales

🎨 Interfaz Moderna

Diseño responsive basado en AdminLTE, accesible desde cualquier dispositivo

🔒 Seguridad

Sistema de autenticación robusto con soporte para Google OAuth

👥 Gestión de Roles

Control de permisos granular según el rol del usuario

📊 Dashboard Interactivo

Estadísticas en tiempo real con gráficos visuales

📦 Control de Inventario

Control de almacenes y ubicaciones de productos

🔐 Acceso al Sistema

Inicio de Sesión

  • Acceda a la URL del sistema proporcionada por su organización
  • Ingrese su correo electrónico en el campo correspondiente
  • Ingrese su contraseña
  • Complete la verificación reCAPTCHA

📚 Guía de Uso Rápida

Cómo Gestionar Roles

  • Navegue a Menú > Roles
  • Haga clic en "Agregar Rol" para crear uno nuevo
  • Complete el formulario con nombre y descripción
  • Haga clic en "Guardar"
  • Use los botones "Editar" o "Eliminar" para modificar roles existentes

Cómo Gestionar Empleados

  • Navegue a Menú > Empleados
  • Haga clic en "Agregar Empleado"
  • Complete: Nombre, Correo, Teléfono y Rol
  • Haga clic en "Guardar"
  • Use "Editar" o "Eliminar" según necesite

Cómo Registrar Movimientos

  • Navegue a Menú > Movimientos
  • Haga clic en "Agregar Movimiento"
  • Seleccione: Producto, Ubicación, Empleado
  • Elija el tipo: Entrada o Salida
  • Ingrese la cantidad
  • Haga clic en "Guardar"
CONSEJO: Siempre verifique el inventario actual antes de registrar una salida para asegurar que haya stock disponible.

❓ Preguntas Frecuentes

¿Qué hago si olvidé mi contraseña?

Contacte al administrador del sistema para restablecerla.

¿Por qué no puedo eliminar un movimiento?

Solo los administradores pueden eliminar movimientos para mantener la integridad del inventario.

¿Cómo sé si un producto tiene stock bajo?

El Dashboard muestra un indicador "Stock Bajo" en rojo cuando los productos están por debajo del mínimo establecido.

¿Puedo exportar los datos?

Sí, las tablas incluyen opciones de exportación a diferentes formatos.

¿El sistema funciona en dispositivos móviles?

Sí, el sistema tiene diseño responsive y funciona perfectamente en tablets y smartphones.
